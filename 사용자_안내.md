# 법인 영업 솔루션 - 사용 방법 (Windows)

> 개발 프로그램은 설치하지 않아도 됩니다.  
> 받은 파일만 압축 풀고 실행 파일 더블클릭하면 됩니다.

---

## 1. 다운로드

- **보내준 링크**에서 **ZIP 파일**을 받습니다.
- 예: `법인영업솔루션_윈도우.zip` 같은 이름의 파일을 **다운로드** 버튼으로 받으세요.

---

## 2. 압축 풀기

- 다운로드한 ZIP 파일을 **우클릭** → **압축 풀기** (또는 "모두 압축 풀기")를 선택합니다.
- **풀어 둘 위치**: 바탕화면이나 문서 폴더 등 **원하는 곳**을 선택하면 됩니다.
- 압축이 풀리면 **법인영업솔루션** 이라는 이름의 **폴더**가 생깁니다.

---

## 3. 실행

1. **법인영업솔루션** 폴더를 연다.
2. 그 안에 있는 **법인영업솔루션.exe** 파일을 **더블클릭**한다.
3. **검은 창(명령 프롬프트)** 이 뜨면 **닫지 말고** 그대로 둔다.
4. 잠시 후 **브라우저(크롬 등)** 가 자동으로 열린다.

---

## 4. 사용 순서 (브라우저 화면)

1. **검색 키워드** 입력 (예: 마스크팩)
2. **정렬** 선택 (기본값 그대로 둬도 됨)
3. **수집할 개수** 입력 (예: 10 → 중복 제외 10개까지 수집)
4. **시작하기** 버튼 클릭
5. 새로 뜬 **브라우저 창**에서 **네이버 로그인** (아이디·비밀번호, 캡차 등)
6. 로그인 후 웹 페이지로 돌아와서 **로그인 완료했어요 → 다음** 클릭
7. 검색 화면이 보이면 **보안 확인 완료했어요 → 수집 시작** 클릭
8. 수집이 끝나면 **엑셀 파일 다운로드** 버튼으로 저장

저장된 엑셀에는 **회사이름, 사업자등록번호, 회사이메일주소**가 들어 있습니다.

---

## 5. 주의사항

- **검은 창을 닫으면** 프로그램이 종료됩니다. 사용하는 동안에는 그대로 두세요.
- **법인영업솔루션 폴더 안의 파일·폴더**는 지우거나 옮기지 마세요. 폴더 전체를 그대로 두고 사용하세요.
- Python, Visual Studio 등 **별도 프로그램 설치는 필요 없습니다.**

---

## 문제가 생겼을 때

- **exe를 더블클릭해도 아무 반응이 없을 때**  
  → 폴더 안의 다른 파일을 건드리지 않았는지 확인하고, **폴더 전체**를 다시 받아서 압축 풀어 보세요.
- **검은 창이 바로 사라질 때**  
  → 바이러스 차단 프로그램이 실행을 막는지 확인해 보세요. 필요하면 “허용”으로 설정하거나, 보내준 사람에게 문의하세요.
- **브라우저가 안 열릴 때**  
  → 검은 창이 떠 있는 상태에서 1~2분 기다려 보세요. 그래도 안 되면 PC를 한 번 재시작한 뒤 다시 실행해 보세요.

궁금한 점은 **프로그램을 보내준 사람**에게 문의하시면 됩니다.
